# MiniCursor

A modular local AI coding assistant for learning and experimenting with modern LLM systems.

## Goals

* Support multiple inference backends (MLX and `llama.cpp`)
* Build a local coding assistant with RAG and agent capabilities
* Explore KV cache optimization techniques for edge LLM inference
* Provide a flexible platform for benchmarking and research

## Planned Features

* Local LLM inference
* MLX backend
* `llama.cpp` backend
* Retrieval-Augmented Generation (RAG)
* ReAct-based agent
* Tool calling
* KV cache management
* Performance benchmarking
